Lookup Types

A lookup entity provide a set of valid values for a given entity. For Example: this can be used to fill drop-downs with values.

Resource Path: lookup/{entity}

Resource Query Parameters:

Parameter Required Defaults
max false 10000
sortField false description
sortOrder false ASC
offset false 0
status false A
details false false
